# Claude Code Skills

Custom skills for Claude Code.

## Skills

| Skill | Description |
|-------|-------------|
| branch-audit | Daily branch audit across remote dev machines, syncing git progress with Jira |
| cad | Script-driven CAD generation, inspection, and validation (STEP, STL, DXF, GLB) |
| consolidate-memory | Reflective pass over memory files — merge duplicates, fix stale facts, prune index |
| docx | Word document creation, editing, and manipulation |
| fix-jira-bug | End-to-end workflow for picking up and fixing Jira bugs |
| paper-code-checker | Daily check for code implementations of tracked research papers |
| pdf | PDF processing — extract, merge, split, create, fill forms |
| pptx | PowerPoint presentation creation and editing |
| pr-reviewer | Automated PR review bot via GitHub Actions |
| schedule | Create or update scheduled tasks |
| scholar-digest | Google Scholar Alert digest processor, posts to Slack |
| scholar-inbox | Scholar Inbox digest summarizer via Chrome or Gmail |
| setup-cowork | Guided Cowork setup — install plugins, connect tools |
| skill-creator | Create, modify, and benchmark skills |
| weekly-training-optimizer | Weekly CADAssembliesCrafter training optimization on Scaleway |
| xlsx | Excel spreadsheet creation, editing, and analysis |
